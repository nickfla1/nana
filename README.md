## Nana
Fast and customizable Node.js package manager written in Rust!

## Usage

#### Print Nana's version
```sh
nana version
```

#### Install dependencies in the current directly
```sh
nana install
```

#### Run custom script from `package.json`
```sh
nana <script-name>
```
